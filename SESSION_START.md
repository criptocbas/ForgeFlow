# Quick Start for New Grok Session

1. Read these files in order:
   - AGENTS.md (this project's full context, goals, standards, current state)
   - README.md (project vision and details)
   - docs/ARCHITECTURE.md (tech + diagram)

2. Confirm understanding of:
   - ForgeFlow = real-time perps strategies on FlashTrade + MagicBlock ER (target +50% prize boost)
   - Focus: code quality, professionalism, deep correct integration of ER + FlashTrade
   - VPS vs laptop considerations
   - Pure scripts in `scripts/` are high value

3. Ask the user for the specific next task.

This combination should make any new session feel continuous.
